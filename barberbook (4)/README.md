O BarberBook foi desenvolvido com o objetivo de auxiliar barbeiros autônomos no gerenciamento de atendimentos, permitindo maior organização dos horários, clientes e serviços realizados.

O sistema busca oferecer uma interface simples, moderna e funcional para facilitar o controle da rotina da barbearia.

▶️ Como Executar o Projeto
1. Instalar dependências

Abra o terminal na pasta do projeto:

npm install
2. Executar o Frontend
npm run dev

A aplicação ficará disponível em:

http://localhost:5173
3. Executar o Backend

Em outro terminal:

npm run server
